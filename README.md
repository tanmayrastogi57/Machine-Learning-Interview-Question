## Basic Concepts

1. **What is the difference between supervised and unsupervised learning?**
   - **Supervised Learning**: Involves training a model on labeled data, where the desired output is known. Example algorithms include linear regression, decision trees, and support vector machines.
   - **Unsupervised Learning**: Involves training a model on unlabeled data, where the model tries to find patterns or groupings without predefined labels. Example algorithms include k-means clustering and principal component analysis (PCA).

2. **Explain the bias-variance tradeoff.**
   - **Bias**: Error due to overly simplistic models that fail to capture the underlying trends (underfitting).
   - **Variance**: Error due to overly complex models that capture noise as if it were part of the trend (overfitting).
   - **Tradeoff**: Balancing bias and variance is crucial for building models that generalize well to new data.

## Algorithms and Models

3. **What is regularization, and why is it useful?**
   - **Regularization**: Techniques used to prevent overfitting by penalizing large coefficients in the model. Common methods include L1 (Lasso) and L2 (Ridge) regularization.
   - **Usefulness**: Helps improve the modelâ€™s performance on unseen data by reducing its complexity.

4. **Describe how a decision tree works.**
   - **Mechanism**: A decision tree splits the data into subsets based on feature values, creating branches that lead to decision nodes or leaf nodes. The goal is to create branches that reduce entropy or increase information gain.

## Evaluation Metrics

5. **What is the difference between precision and recall?**
   - **Precision**: The ratio of true positive predictions to the total positive predictions. Measures the accuracy of positive predictions.
   - **Recall**: The ratio of true positive predictions to the total actual positives. Measures the ability to capture all relevant instances.

6. **Explain cross-validation and why it is important.**
   - **Cross-Validation**: A technique to assess how well a model generalizes to an independent dataset by splitting the data into training and validation sets multiple times. The most common method is k-fold cross-validation.
   - **Importance**: Helps in detecting overfitting and provides a more accurate estimate of model performance.

## Advanced Topics

7. **What is the difference between bagging and boosting?**
   - **Bagging**: Combines multiple models (typically of the same type) trained on different subsets of the data (with replacement) to reduce variance. Example: Random Forest.
   - **Boosting**: Sequentially trains models, each trying to correct the errors of its predecessor to reduce bias and variance. Example: AdaBoost, Gradient Boosting.

8. **Describe the concept of a neural network.**
   - **Neural Network**: A computational model inspired by the human brain, consisting of layers of interconnected nodes (neurons). Each connection has a weight, and the network learns by adjusting these weights based on the error of the output compared to the target.

## Practical Implementation

9. **How would you handle missing data in a dataset?**
   - **Methods**: Imputation (mean, median, mode), deletion, or using algorithms that support missing values.
